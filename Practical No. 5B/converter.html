Programs to convert Celsius into Fahrenheit

Prelogue:

Temprature can be measured by different units. Two of them are Fahrenheit(0F) and Celsius(0C).

Here are some programs to convert Celsius to Fahrenheit by using the scientific formula in different languages.


Code 

            C :  
            
                #include < stdio.h >
                int main()  {   
                    float fahrenheit, celsius;  
                    scanf("%d", &celsius);
                    fahrenheit =( (celsius*9)/5)+32;  
                    printf("\n\n Temperature in fahrenheit is:  %f",fahrenheit);  
                    return (0);
                }
            
            Python : 
            
                celsius = 37.5
                fahrenheit = (celsius * 1.8) + 32
                print('%0.1f degree Celsius is equal to %0.1f degree Fahrenheit' %(celsius,fahrenheit))
            
            Java :
            
                public class temperature  {  
                public static void main (String args[]) { 
                        float Fahrenheit, Celsius;  
                        Celsius= 13;  
                        Fahrenheit =((Celsius*9)/5)+32;  
                        System.out.println("Temperature in Fahrenheit is: "+Fahrenheit);  
                    }}
